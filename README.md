# Study_plan
УЧЕБНЫЙ ПЛАН

Программа реализует функционал для работы с данными учебного плана.

Возможности:
- Чтение информации из текстового файла disciplines.txt;
- Проверка существования файла;
- Три вида отчетов с сортировкой по разным критериям

Отчеты:
- Полный список дисциплин;
- Отчёт по виду отчётности;
- Отчёт по диапазону часов;
- Выход из программы.
